An app to handle internal poker league. Shows all played seasons, best players, standings, schedules, rules and more. Check it out!

### Build with

- React
- TypeScript
- Vite
- TailwindCSS
- Zustand

### Setup

`yarn install`

### Dev

`yarn dev && yarn build-css-watch`

### Deploy

```
Pull request to PRO and Github Action will run
```
